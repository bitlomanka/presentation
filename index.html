Используем WATERMARK
  • Шаг 1.
    Выбираем исходное избражение.
  • Шаг 2.
    Выбираем водяной знак.
  • Шаг 3.
    Выбираем положение водяного знака. Есть два варианта отображения, которые настраиваются по осям " X " и " Y ".
  • Шаг 4.
    Настраиваем прозрачность водяного знака.
  • Шаг 5.
    Скачиваем готовое изображение.
